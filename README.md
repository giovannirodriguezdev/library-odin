# Library Project - The Odin Project

## 🚀 Features
- Add new books with title, author, pages, and read status
- Toggle read status of books
- Remove books from the collection
- Responsive UI using modern CSS and JavaScript

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
